# AyBook

AyBook is a Flutter app for managing a collection of books using Firebase Firestore.

## Features

- Splash screen with logo and app name
- Menu screen with navigation
- Add Book screen (Book ID, Name, Author, Publisher, Date Published)
- Retrieve Books screen (list all books from Firebase)

## Structure

- `lib/models/book.dart`: Book data model
- `lib/services/book_service.dart`: Firebase Firestore service
- `lib/screens/`: All UI screens (Splash, Menu, Add Book, Retrieve)

## Setup

1. Run `flutter pub get` to install dependencies.
2. Set up Firebase for your app (add `google-services.json` or `GoogleService-Info.plist`).
3. Run the app on your device/emulator.

## Requirements

- Flutter 3.x
- Firebase account and project
